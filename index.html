Overview

The Industrial Visit and Planning System is a web based reservation system designed to make it easier for students to book visits to various industrial locations at specific times.

Target Audiences

Admin

The Admin is simply an individual that can view all the visits planned by the students as well as make changes if required. Admins can access the modules such as adding subjects, places, package details.

Login

Students

If you're a student looking to visit a particular Work environment, this place is the perfect location because may login and view various industrial tour and travel destinations with the package details. Click the started button to create an account with us.
